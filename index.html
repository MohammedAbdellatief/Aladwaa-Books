شرح بأسلوب مُبسط، سهل وممتع

طريقة شرح مبسطة، واستخدام أحدث الطرق التربوية لتسهيل المعلومات وتيسيرها؛ بحيث يستمتع الطالب بطريقة العرض التى تتميز برسومات وأشكال توضيحية بعيدًا عن الإيجاز أو الإسهاب فى الشرح

أول كتاب تعليمى بتقنية QR Code

نقدم لكم تقنية الـQR Code التى تصل بالطالب لأفضل مستويات التعلم؛ إذ تتيح له مشاهدة فيديوهات تعليمية، وحل تدربيات تفاعلية على الدرس من هاتفه مباشرةً.

  • شاهد علامة الـQR Code داخل الدرس
  • استخدم تطبيق "الأضواء" لقراءة الـ QR Code
  • استمتع بالفيديو أو قم بحل تدريبات تفاعلية مباشرةً
